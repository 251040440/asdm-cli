# ASDM CLI Tools Repository

This repository contains the source code for the ASDM CLI tools, which are designed to help developers set up and manage their development environment for ASDM (AI First System Development Methodology & Platform) projects.

## Repository Structure

- **[asdm-bootstrapper-cli](./asdm-bootstrapper-cli/)** - CLI tool for downloading and configuring ASDM library resources (toolsets, contexts, and specs)

## Getting Started

Each CLI tool has its own documentation. Please navigate to the respective directory for detailed information:

- See [asdm-bootstrapper-cli/README.md](./asdm-bootstrapper-cli/README.md) for bootstrapper usage and installation
